### Embed a playback iFrame?
Add this snippet to your html:

```
<iframe src='https://playback.syrf.io/?token=YOUR_DEVELOPER_TOKEN&race=RACE_ID'>
```
:::tip Reminder

Don't forget to replace your token and the race ID.
